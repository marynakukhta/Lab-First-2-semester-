Glossary
class User - клас, який описує користувача системи. Відповідає за інформацію про нього.
class Contact - клас, який зберігає інформацію про кожного контакта користувача.
class ContactList - клас, який зберігає список контактів  та деяку інформацію про кожного.
class Call - клас, який відповідає за дзвінки.
class Chat - клас, що відповідає за чат, його функціональність. 
class NormalChat - потомок класу Chat, відповідає за чат з одним із контактів, його функціональність. 
class GroupChat -  потомок класу Chat , відповідає за чат з групою контактів, його функціональність. 
class BotChat -  потомок класу Chat , відповідає за чат з ботом, його функціональність. 
class Message - клас,  що відповідає за повідомлення, іх функціональність, інформацію.
class VoiceMessage - потомок класу Message, відповідає за голосові повідомлення.
class VideoMessage -  потомок класу Message, відповідає за відео повідомлення.  
class Media - класс, що відповідає за можлвість перегляду відправлених/відправлення медіа документів іншим контактам.
-----------------------------------------------------------------------------------------------------------------
class Location - клас, який відповідає за можливість пошуку об’єкта за локацією, відправку локації, знаходження координат користувача.
Class Message -  клас, який відповідає за відправку локацій.
-----------------------------------------------------------------------------------------------------------------
User1 - користувач системою. 
User2 - контакт з яким йде листування.
